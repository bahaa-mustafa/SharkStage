# SharkStage Frontend

## المتطلبات
- Node.js 18 أو أحدث
- إعداد ملف البيئة `.env.local` بالقيم التالية:
  - `NEXT_PUBLIC_API_URL` يشير إلى خادم الـ API (مثل `http://localhost:5000`)
  - `NEXT_PUBLIC_GOOGLE_CLIENT_ID` رقم تعريف تطبيق Google OAuth

## التثبيت والتشغيل
1. `npm install`
2. `npm run dev`
3. افتح `http://localhost:3000`

## ملاحظات التوثيق
- الواجهة تستخدم Redux لإدارة حالة المستخدم (`authSlice`).
- يتم تحميل التوكن داخل Cookie من الخادم، و`checkAuth` تتحقق منه عند بدء التطبيق.

## اختبار سيناريوهات الدخول والتسجيل
1. **تسجيل حساب يدوي**
   - انتقل إلى `/sign/up`
   - أكمل الحقول الإلزامية (تشمل تأكيد كلمة السر)
   - جرّب إدخال بيانات خاطئة للتأكد من ظهور الرسائل الفورية
2. **دخول بحساب مسجل**
   - انتقل إلى `/sign/in`
   - اختبر التحذيرات عند ترك الحقول فارغة أو إدخال بريد غير صحيح
3. **الدخول عبر Google**
   - تأكد من تهيئة `NEXT_PUBLIC_GOOGLE_CLIENT_ID`
   - في صفحة التسجيل اختر نوع الحساب ثم اضغط "Google"
   - في صفحة تسجيل الدخول استخدم Google لحساب موجود مسبقًا

للتأكد من نجاح الإعداد، راقب وحدة التحكم في المتصفح للتأكد من عدم وجود أخطاء، وتحقق من cookies للتأكد من استقبال التوكن بعد أي عملية تسجيل/دخول.
